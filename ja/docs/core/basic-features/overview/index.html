基本機能の概要

AIBuds SDK の基本機能ドキュメントへようこそ。このセクションでは、AIBuds デバイスで実行できる基本的な操作について説明します。

使用可能な機能​

📄️出荷時設定にリセット

デバイスを出荷時設定に復元し、すべてのユーザーデータとカスタム設定を消去します

📄️電源オフ

デバイスをプログラムで電源オフする

📄️デバイスのペア解除

接続されたデバイスとのペアリング関係を解除します

はじめに​

AIBuds SDK の使用を開始するには、以下の手順に従ってください：

  1. SDK をインストール - AIBuds.xcframework を iOS プロジェクトに統合します
  2. SDK を初期化 - API 資格情報を使用して SDK を設定します
  3. デバイスに接続 - AIBuds デバイスとの接続を確立します
  4. 機能を探索 - このセクションのドキュメントに記載されている API を使用してデバイスと対話します

重要な概念​

デバイス管理​

  • 電源管理：power-off を使用してデバイスの電源状態を制御します
  • ペアリング管理：unpair-device を使用してデバイスのペアリング関係を管理します
  • ファクトリリセット：factory-reset を使用してデバイスを工場出荷時設定にリセットします

エラー処理​

すべての操作には包括的なエラー処理が含まれています。特定のエラーコードと復旧方法については、各 API のドキュメントを参照してください。

プラットフォームの考慮事項​

iOS の Bluetooth ペア解除制限など、プラットフォーム固有の制限に注意してください。各機能ドキュメントには、該当する場合にプラットフォーム固有の注意事項が含まれています。
