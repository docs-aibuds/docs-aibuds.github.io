Cihazın Eşlemesini Kaldır

Cihazın eşlemesini kaldırma işlemi, cihaz ile bağlı cihaz arasındaki eşleme ilişkisini kaldırır. Bu işlem, cihazı kalıcı olarak bağlantısını kesmeniz veya başka bir cihazla eşleme hazırlamanız gerektiğinde kullanışlıdır.

Ön Koşullar​

Eşleme kaldırmadan önce şunlardan emin olun:

  • Cihaz bağlı ve kararlı bir durumda
  • Gerekli veri senkronizasyonu tamamlandı
  • Kullanıcı, eşleme kaldırma işleminden sonra cihazın bağlantısının kesileceğini anlıyor

API Referansı​

Çerçeve​

AIBuds.xcframework

İçe Aktarma​

SDK'yı kullanmak istediğiniz dosyalarda ana başlığı içe aktarın:

  • Swift
  • Objective-C
import AIBuds
#import <AIBuds/AIBuds.h>
#import <AIBuds/AIBuds-Swift.h>

Protokol​

unpair metodu aşağıdaki protokollü tanımlanmıştır. Protokol, temel cihaz API protokolünden miras alır.

  • Swift
  • Objective-C
/// Eşleme kaldırma dahil ortak cihaz işlemlerini tanımlar
protocol DeviceCommonAPI: DeviceAPI {
    /// Cihazın eşlemesini kaldırır
    /// - Parameters:
    ///   - completion: İşlem sonucunu döndüren tamamlama geri çağrısı
    ///     - success: İşlem başarılıysa `true`; aksi takdirde `false`.
    ///     - error: Oluşan hatayı açıklayan bir `NSError` nesnesi veya işlem başarılıysa `nil`.
    func unpair(_ completion: AIBudsCompletionHandler?)
}
/// Eşleme kaldırma dahil ortak cihaz işlemlerini tanımlar
@protocol AIBudsDeviceCommonAPI <AIBudsDeviceAPI>

    /// - Parameters:
    ///   - completion: İşlem sonucunu döndüren tamamlama geri çağrısı
    ///     - success: İşlem başarılıysa `true`; aksi takdirde `false`.
    ///     - error: Oluşan hatayı açıklayan bir `NSError` nesnesi veya işlem başarılıysa `nil`.
    - (void)unpairWithCompletion:(AIBudsCompletionHandler)completion;
@end

Parametreler​

ParametreTürAçıklama
completionAIBudsCompletionHandler?İşlem tamamlandığında çağrılan isteğe bağlı tamamlama geri çağrısı

Geri Çağrı Parametreleri:

Dönüş Değeri​

Bu metot doğrudan bir değer döndürmez. Sonuç, tamamlama geri çağrısı aracılığıyla sağlanır.

Kullanım Örnekleri​

  • Swift
  • Objective-C
import AIBuds

class DeviceManager {

    /// Bağlı cihaz
    weak var device: DeviceConvertible?

    /// Bağlı cihazın eşlemesini kaldırır
    func unpairDevice() {
        // Cihazın eşleme kaldırma protokolünü desteklediğinden emin olun
        guard let device = device as? DeviceCommonAPI else {
            print("Cihaz eşleme kaldırmayı desteklemiyor")
            return
        }

        // Tamamlama işleyicisiyle eşleme kaldırmayı yürütün
        device.unpair { [weak self] success, error in
            // Hata durumunu işleyin
            if !success {
                let errorMessage = {
                    if let error = error {
                        return "\(error)"
                    }
                    return "Bilinmeyen hata"
                }()
                print("Eşleme kaldırma başarısız: \(errorMessage)")
                return
            }
            // Başarı durumunu işleyin
            print("Eşleme kaldırma başarıyla tamamlandı")
        }
    }
}
#import <AIBuds/AIBuds.h>

@interface DeviceManager ()

/// Bağlı cihaz
@property (weak, nonatomic) id<AIBudsDeviceConvertible> device;

@end

@implementation DeviceManager

- (void)unpairDevice {
    __weak typeof(self) weakSelf = self;


    id<AIBudsDeviceCommonAPI> device = (id<AIBudsDeviceCommonAPI>)self.device;
    // Cihazın eşleme kaldırma protokolünü desteklediğinden emin olun
    if ([device conformsToProtocol:@protocol(AIBudsDeviceCommonAPI)]) {
        // Tamamlama işleyicisiyle eşleme kaldırmayı yürütün
        [device unpairWithCompletion:^(BOOL success, NSError * _Nullable error) {
            // Hata durumunu işleyin
            if (!success) {
                NSLog(@"Eşleme kaldırma başarısız: %@", error);
                return;
            }
            // Başarı durumunu işleyin
            NSLog(@"Eşleme kaldırma başarıyla tamamlandı");
        }];
    }
}

@end

En İyi Uygulamalar​

  1. Kullanıcıyı Onaylayın: Eşleme kaldırmayı başlatmadan önce her zaman bir onay iletişim kutusu görüntüleyin, çünkü bu işlem cihazın bağlantısını kesecek ve yeniden eşleme gerektirecektir.

  2. Arka Plan Yürütmeyi İşleyin: UI'yi güncellerken tamamlama işleyicisini bir DispatchQueue.main.async bloğuna sarın.

  3. Zayıf Self Referansı: Retain döngülerini önlemek için tamamlama işleyicisinde [weak self] kullanın.

  4. Protokol Uyumluluğunu Kontrol Edin: Metodu çağırmadan önce cihazın DeviceCommonAPI protokolüne uyduğunu doğrulayın.

  5. Bağlantı Kesmeyi İşleyin: Başarılı bir eşleme kaldırma işleminden sonra, cihaz bağlantısını düzgün bir şekilde işleyin ve yeniden eşleme için rehberlik sağlayın.

Platform Sınırlamaları​

iOS Sistem Bluetooth Sınırlaması​

iOS'ta, sistem Bluetooth ayarları uygulamaların Bluetooth cihazlarının eşlemesini programatik olarak kaldırmasına izin vermez. Bu, Apple tarafından güvenlik ve kullanıcı kontrolü için uygulanan sistem düzeyinde bir kısıtlamadır.

Bu ne anlama geliyor:

  • unpair çağırdıktan sonra bile, iOS cihaz için BLE eşleme bilgilerini saklayacaktır
  • Uygulama yeniden başlatıldığında veya Bluetooth etkinleştirildiğinde cihaz otomatik olarak yeniden bağlanabilir
  • Cihaz iOS Ayarlar > Bluetooth'ta görünür kalacaktır

Önerilen Kullanıcı Rehberliği:

iOS uygulamanızda eşleme kaldırma işlevini uygularken, kullanıcıları iOS Ayarlarından manuel olarak eşleme kaldırmaya yönlendirmeniz gerekir:

  1. Tam eşleme kaldırma için: Kullanıcıları Ayarlar > Bluetooth'a gidin, cihazı bulun, "i" simgesine dokunun ve "Bu Cihazı Unut" seçeneğini seçin
  2. Açık UI geri bildirimi sağlayın: Kullanıcı eşleme kaldırmayı istediğinde talimatları veya Bluetooth ayarlarına derin bağlantıyı gösterin
  3. Uygulama düzeyinde bağlantı kesme: unpair metodu hala uygulamanız açısından cihazın bağlantısını kesecektir, ancak sistem eşlemesi kalacaktır
  • Swift
  • Objective-C
/// Kullanıcıyı iOS Ayarlarından eşleme kaldırmaya yönlendirir
func promptUserToUnpairFromSettings() {
    // Talimatlarla uyarı göster
    let alert = UIAlertController(
        title: "Cihazın Eşlemesini Kaldır",
        message: "Cihazın eşlemesini tamamen kaldırmak için lütfen Ayarlar > Bluetooth'a gidin, cihazınızın yanındaki 'i' simgesine dokunun ve 'Bu Cihazı Unut' seçeneğini seçin.",
        preferredStyle: .alert
    )
    alert.addAction(UIAlertAction(title: "Ayarları Aç", style: .default) { _ in
        // Bluetooth ayarlarına derin bağlantı
        if let url = URL(string: "App-prefs:Bluetooth") {
            UIApplication.shared.open(url)
        }
    })
    alert.addAction(UIAlertAction(title: "İptal", style: .cancel))
    
    // Uyarıyı sun
    if let viewController = UIApplication.shared.windows.first?.rootViewController {
        viewController.present(alert, animated: true)
    }
}
/// Kullanıcıyı iOS Ayarlarından eşleme kaldırmaya yönlendirir
- (void)promptUserToUnpairFromSettings {
    UIAlertController *alert = [UIAlertController 
        alertControllerWithTitle:@"Cihazın Eşlemesini Kaldır"
        message:@"Cihazın eşlemesini tamamen kaldırmak için lütfen Ayarlar > Bluetooth'a gidin, cihazınızın yanındaki 'i' simgesine dokunun ve 'Bu Cihazı Unut' seçeneğini seçin."
        preferredStyle:UIAlertControllerStyleAlert];
    
    [alert addAction:[UIAlertAction actionWithTitle:@"Ayarları Aç" style:UIAlertActionStyleDefault handler:^(UIAlertAction * _Nonnull action) {
        NSURL *url = [NSURL URLWithString:@"App-prefs:Bluetooth"];
        if (url && [[UIApplication sharedApplication] canOpenURL:url]) {
            [[UIApplication sharedApplication] openURL:url options:@{} completionHandler:nil];
        }
    }]];
    
    [alert addAction:[UIAlertAction actionWithTitle:@"İptal" style:UIAlertActionStyleCancel handler:nil]];
    
    UIViewController *viewController = [UIApplication sharedApplication].windows.firstObject.rootViewController;
    [viewController presentViewController:alert animated:YES completion:nil];
}

Notlar​

  • Eşleme kaldırma komutu çalıştırıldıktan sonra cihazın bağlantısı kesilecektir
  • Cihazın yeniden bağlanması için yeniden eşleme gerekecektir
  • Tüm eşleme bilgileri her iki cihazdan da kaldırılacaktır
  • Devam eden işlemler yarıda kesilecektir
  • Eşleme kaldırma işlemi birkaç saniye sürebilir
