Streaming Recognition

Perform real-time speech-to-text streaming recognition for live captioning and transcription.

Prerequisites​

Before calling this API, ensure:

  • The device is connected
  • An AI provider has been selected
  • Network connectivity is available
  • The device supports streaming recognition

API Reference​

Framework​

AIBuds.xcframework

Import​

  • Swift
  • Objective-C
import AIBuds
#import <AIBuds/AIBuds.h>
#import <AIBuds/AIBuds-Swift.h>

Protocol​

The streaming recognition methods are defined in the following protocol:

  • Swift
  • Objective-C
protocol StreamingRecognitionAPI: DeviceAPI {
    func startStreamingRecognition(language: String, 
                                  delegate: StreamingRecognitionDelegate?)
    func stopStreamingRecognition()
    func isRecognizing() -> Bool
}

protocol StreamingRecognitionDelegate: AnyObject {
    func onRecognitionResult(_ result: String, isFinal: Bool)
    func onRecognitionError(_ error: Error)
    func onRecognitionStarted()
    func onRecognitionStopped()
}
@protocol AIBudsStreamingRecognitionAPI <AIBudsDeviceAPI>
- (void)startStreamingRecognitionWithLanguage:(NSString *)language delegate:(id<AIBudsStreamingRecognitionDelegate>)delegate;
- (void)stopStreamingRecognition;
- (BOOL)isRecognizing;
@end

@protocol AIBudsStreamingRecognitionDelegate <NSObject>
- (void)onRecognitionResult:(NSString *)result isFinal:(BOOL)isFinal;
- (void)onRecognitionError:(NSError *)error;
- (void)onRecognitionStarted;
- (void)onRecognitionStopped;
@end

Parameters​

ParameterTypeDescription
languageStringLanguage code for recognition (e.g., "en-US", "zh-CN").
delegateStreamingRecognitionDelegate?Delegate for receiving recognition results.

Usage Examples​

  • Swift
  • Objective-C
import AIBuds

class StreamingRecognitionManager: NSObject, StreamingRecognitionDelegate {
    weak var device: DeviceConvertible?
    
    func startRecognition() {
        guard let device = device as? StreamingRecognitionAPI else {
            print("Device does not support StreamingRecognitionAPI")
            return
        }
        
        if device.isRecognizing() {
            print("Already recognizing")
            return
        }
        
        device.startStreamingRecognition(language: "en-US", delegate: self)
    }
    
    func stopRecognition() {
        guard let device = device as? StreamingRecognitionAPI else {
            return
        }
        device.stopStreamingRecognition()
    }
    
    // MARK: - StreamingRecognitionDelegate
    
    func onRecognitionResult(_ result: String, isFinal: Bool) {
        print("Recognition result: \(result) (final: \(isFinal))")
    }
    
    func onRecognitionError(_ error: Error) {
        print("Recognition error: \(error)")
    }
    
    func onRecognitionStarted() {
        print("Recognition started")
    }
    
    func onRecognitionStopped() {
        print("Recognition stopped")
    }
}
#import <AIBuds/AIBuds.h>

@interface StreamingRecognitionManager () <AIBudsStreamingRecognitionDelegate>
@property (weak, nonatomic) id<AIBudsDeviceConvertible> device;
@end

@implementation StreamingRecognitionManager

- (void)startRecognition {
    id<AIBudsStreamingRecognitionAPI> device = (id<AIBudsStreamingRecognitionAPI>)self.device;
    if (![device conformsToProtocol:@protocol(AIBudsStreamingRecognitionAPI)]) {
        NSLog(@"Device does not support StreamingRecognitionAPI");
        return;
    }
    
    if ([device isRecognizing]) {
        NSLog(@"Already recognizing");
        return;
    }
    
    [device startStreamingRecognitionWithLanguage:@"en-US" delegate:self];
}

- (void)stopRecognition {
    id<AIBudsStreamingRecognitionAPI> device = (id<AIBudsStreamingRecognitionAPI>)self.device;
    [device stopStreamingRecognition];
}

#pragma mark - AIBudsStreamingRecognitionDelegate

- (void)onRecognitionResult:(NSString *)result isFinal:(BOOL)isFinal {
    NSLog(@"Recognition result: %@ (final: %d)", result, isFinal);
}

- (void)onRecognitionError:(NSError *)error {
    NSLog(@"Recognition error: %@", error);
}

- (void)onRecognitionStarted {
    NSLog(@"Recognition started");
}

- (void)onRecognitionStopped {
    NSLog(@"Recognition stopped");
}

@end

Best Practices​

  1. Handle Network: Ensure stable network connection for streaming.
  2. Use Weak References: Avoid retain cycles with delegates.
  3. Manage State: Track recognition state to avoid duplicate starts.

Notes​

  • Requires active network connection for cloud-based recognition
  • Latency depends on network conditions and AI provider
  • Supported languages vary by provider
