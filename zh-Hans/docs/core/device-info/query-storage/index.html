Query Storage

Retrieve information about the device's storage capacity and current usage.

Prerequisites​

Before calling this API, ensure:

  • The device is connected and in a stable state

API Reference​

Framework​

AIBuds.xcframework

Import​

  • Swift
  • Objective-C
import AIBuds
#import <AIBuds/AIBuds.h>
#import <AIBuds/AIBuds-Swift.h>

Protocol​

The queryStorage method is defined in the following protocol:

  • Swift
  • Objective-C
protocol DeviceInfoAPI: DeviceAPI {
    func queryStorage(_ completion: @escaping (Result<StorageInfo, Error>) -> Void)
}

struct StorageInfo {
    let totalSpace: Int64      // Total storage in bytes
    let usedSpace: Int64       // Used storage in bytes
    let availableSpace: Int64  // Available storage in bytes
}
@protocol AIBudsDeviceInfoAPI <AIBudsDeviceAPI>
- (void)queryStorageWithCompletion:(void(^)(AIBudsStorageInfo * _Nullable info, NSError * _Nullable error))completion;
@end

@interface AIBudsStorageInfo : NSObject
@property (nonatomic, assign) int64_t totalSpace;
@property (nonatomic, assign) int64_t usedSpace;
@property (nonatomic, assign) int64_t availableSpace;
@end

Parameters​

ParameterTypeDescription
completion(Result<StorageInfo, Error>)Completion callback that returns storage information or an error.

Usage Examples​

  • Swift
  • Objective-C
import AIBuds

class DeviceManager {
    weak var device: DeviceConvertible?
    
    func checkStorage() {
        guard let device = device as? DeviceInfoAPI else {
            print("Device does not support DeviceInfoAPI")
            return
        }
        
        device.queryStorage { result in
            switch result {
            case .success(let info):
                let totalGB = info.totalSpace / (1024 * 1024 * 1024)
                let availableGB = info.availableSpace / (1024 * 1024 * 1024)
                print("Total: \(totalGB) GB, Available: \(availableGB) GB")
            case .failure(let error):
                print("Failed to query storage: \(error)")
            }
        }
    }
}
#import <AIBuds/AIBuds.h>

@interface DeviceManager ()
@property (weak, nonatomic) id<AIBudsDeviceConvertible> device;
@end

@implementation DeviceManager

- (void)checkStorage {
    id<AIBudsDeviceInfoAPI> device = (id<AIBudsDeviceInfoAPI>)self.device;
    if ([device conformsToProtocol:@protocol(AIBudsDeviceInfoAPI)]) {
        [device queryStorageWithCompletion:^(AIBudsStorageInfo *info, NSError *error) {
            if (error) {
                NSLog(@"Failed to query storage: %@", error);
                return;
            }
            int64_t totalGB = info.totalSpace / (1024 * 1024 * 1024);
            int64_t availableGB = info.availableSpace / (1024 * 1024 * 1024);
            NSLog(@"Total: %lld GB, Available: %lld GB", totalGB, availableGB);
        }];
    }
}

@end

Error Handling​

Error Domain: AIBudsSDK.ErrorDomain

Error CodeDescriptionRecovery Suggestion
.deviceNotConnectedDevice is not connectedEnsure device is paired and connected
.bleCommandExecFailedDueToTimeoutOperation timed outRetry the operation
.deviceNotSupportStorage query is not supportedCheck device capabilities

Best Practices​

  1. Check Before Recording: Query storage before starting long recordings.
  2. Handle Units: Convert bytes to more readable units (KB, MB, GB).

Notes​

  • Storage values are returned in bytes
  • Some devices may not have internal storage
  • Available space may be less than total space due to system usage
