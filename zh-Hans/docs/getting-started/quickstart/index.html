Quick Start

This guide will help you get started with AIBuds SDK iOS quickly. We'll walk you through the basic setup and show you how to perform common tasks.

Step 1: Initialize the SDK​

First, you need to initialize the AIBuds SDK in your application. This is typically done in your AppDelegate or SceneDelegate:

  • Objective-C
  • Swift
// AppDelegate.m
#import <AIBuds/AIBuds.h>

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {
    // Initialize AIBuds SDK
    [AIBudsManager initializeSDKWithAppKey:@"YOUR_APP_KEY"];

    // Enable logging (optional)
    [AIBudsLog setLogLevel:AIBudsLogLevelDebug];

    return YES;
}
// AppDelegate.swift
import AIBuds

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    // Initialize AIBuds SDK
    AIBudsManager.initializeSDK(withAppKey: "YOUR_APP_KEY")

    // Enable logging (optional)
    AIBudsLog.setLogLevel(.debug)

    return true
}

Step 2: Scan for Devices​

To connect to an AIBuds device, you first need to scan for available devices:

  • Objective-C
  • Swift
// Start scanning for devices
[AIBudsManager.shared startScan:^(NSArray<AIBudsDevice *> *devices) {
    // Handle discovered devices
    for (AIBudsDevice *device in devices) {
        NSLog(@"Found device: %@, RSSI: %d", device.name, device.rssi);
    }
}];

// Stop scanning after 10 seconds
DispatchAfter(10.0, ^{
    [AIBudsManager.shared stopScan];
});
// Start scanning for devices
AIBudsManager.shared.startScan { devices in
    // Handle discovered devices
    for device in devices {
        print("Found device: \(device.name), RSSI: \(device.rssi)")
    }
}

// Stop scanning after 10 seconds
DispatchQueue.main.asyncAfter(deadline: .now() + 10.0) {
    AIBudsManager.shared.stopScan()
}

Step 3: Connect to a Device​

Once you've found a device, you can connect to it:

  • Objective-C
  • Swift
// Connect to a device
[AIBudsManager.shared connectDevice:device completion:^(BOOL success, NSError *error) {
    if (success) {
        NSLog(@"Connected to device successfully");
    } else {
        NSLog(@"Failed to connect: %@", error.localizedDescription);
    }
}];
// Connect to a device
AIBudsManager.shared.connectDevice(device) { success, error in
    if success {
        print("Connected to device successfully")
    } else {
        print("Failed to connect: \(error?.localizedDescription ?? "Unknown error")")
    }
}

Step 4: Send Commands​

Once connected, you can send commands to the device:

  • Objective-C
  • Swift
// Example: Set noise cancellation level
[AIBudsManager.shared setANCLevel:AIBudsANCLevelHigh forDevice:device completion:^(BOOL success, NSError *error) {
    if (success) {
        NSLog(@"ANC level set successfully");
    } else {
        NSLog(@"Failed to set ANC level: %@", error.localizedDescription);
    }
}];
// Example: Set noise cancellation level
AIBudsManager.shared.setANCLevel(.high, for: device) { success, error in
    if success {
        print("ANC level set successfully")
    } else {
        print("Failed to set ANC level: \(error?.localizedDescription ?? "Unknown error")")
    }
}

Step 5: Receive Notifications​

You can register for device notifications to receive updates about the device's status:

  • Objective-C
  • Swift
// Register for device status notifications
[[NSNotificationCenter defaultCenter] addObserver:self
                                         selector:@selector(deviceStatusUpdated:)
                                             name:AIBudsDeviceStatusDidChangeNotification
                                           object:nil];

// Handle device status updates
- (void)deviceStatusUpdated:(NSNotification *)notification {
    AIBudsDevice *device = notification.userInfo[AIBudsDeviceKey];
    NSLog(@"Device status updated: %@", device.status);
}
// Register for device status notifications
NotificationCenter.default.addObserver(
    forName: .AIBudsDeviceStatusDidChange,
    object: nil,
    queue: .main
) { notification in
    if let device = notification.userInfo?[AIBudsDeviceKey] as? AIBudsDevice {
        print("Device status updated: \(device.status)")
    }
}

Step 6: Disconnect from a Device​

When you're done with a device, you should disconnect from it:

  • Objective-C
  • Swift
// Disconnect from a device
[AIBudsManager.shared disconnectDevice:device completion:^(BOOL success, NSError *error) {
    if (success) {
        NSLog(@"Disconnected from device successfully");
    } else {
        NSLog(@"Failed to disconnect: %@", error.localizedDescription);
    }
}];
// Disconnect from a device
AIBudsManager.shared.disconnectDevice(device) { success, error in
    if success {
        print("Disconnected from device successfully")
    } else {
        print("Failed to disconnect: \(error?.localizedDescription ?? "Unknown error")")
    }
}
